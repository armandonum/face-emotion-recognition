clona el repositorio 
```bash
git clone https://github.com/armandonum/face-emotion-recognition.git
```
crea tu entorno virtual 
```bash
python -m venv venv3.10
```
se recominda usar python version  3.10


activa el entorno virtual 
```bash
source venv3.10/bin/activate  # comando  en linux
```


instala las dependecias 
```bash
pip install -r requirements.txt
```

asegurate de que no haya conflictos con ninguna version 


ejecuta la aplicacion 
```bash
python app.py
```
